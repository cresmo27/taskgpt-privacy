Política de Privacidad - Task GPT

Fecha de entrada en vigor: 29 de junio de 2025

La extensión Task GPT no recoge, almacena ni comparte ningún tipo de información personal del usuario. Todo el contenido y las tareas introducidas se almacenan localmente en el navegador del usuario utilizando almacenamiento local de Chrome (storage API).

Finalidad

La única finalidad de esta extensión es permitir a los usuarios gestionar tareas y recordatorios personales relacionados con el uso de ChatGPT. La extensión no interactúa con servidores externos ni recopila información remotamente.

Permisos

  • storage: Necesario para guardar las tareas del usuario en su navegador.
  • notifications: Utilizado para mostrar recordatorios de tareas mediante notificaciones locales (opcional y bajo consentimiento).

Datos de usuario

Task GPT no accede a datos sensibles como nombres, contraseñas, ubicación, historial, ni transmite datos a servidores externos.

Contacto

Para cualquier duda o consulta puedes contactar con el desarrollador a través del correo: taskgpt.extension@gmail.com.

Al utilizar esta extensión, aceptas esta política de privacidad.
